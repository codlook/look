Nasıl Kurulur

GitHub hesabı bile gerekmez. Komutu kopyala, terminale yapıştır.

Adım 1
Listeden seç
Bu sayfada ara, filtrele, kurulum komutunu kopyala.
Adım 2 — Modül
Global kur
Tüm projelerde kullanmak için:
lk module install github.com/codlook/look-modules/jwt
Adım 2 — Paket
Projeye kur
Sadece o projenin pkg/ klasörüne:
lk install github.com/codlook/look-jwt
Adım 3
Kodunda kullan
use ile dahil et, hemen kullanmaya başla.
use jwt

token = jwt_sign({"id": 1}, "secret")
data = jwt_verify(token, "secret")
